# Menu

> Размещен в `modules`.

## Может хранить такую структуру.

```json
{
  "config": {
    "modules": [
      {
        "name": "menu",
        "enable": true,
        "config": {
          "list": [
            {
              "id": 1,
              "pageId": 1,
              "position": 1,
              "enable": true,
              "title": "menu.welcome"
            },
            {
              "id": 2,
              "pageId": 3,
              "position": 3,
              "enable": true,
              "title": "#Driver Application"
            },
          ]
        }
      }
    ]
  }
}
```
