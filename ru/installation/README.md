# Установка

> Текстовый текст для Установка.

**[Radiobuttons]** может хранить такую структуру.

```
"custom": {
	"defaultValue": "yes",
	"options": [
		{
			"value": "yes",
			"text": "#Yes",
			"action": {
				"type": "SHOW_CONTAINER",
				"nextSibling": 1
			}
		},
		{
			"value": "no",
			"text": "#No",
			"action": {
				"type": "HIDE_CONTAINER",
				"nextSibling": 1
			}
		}
	]
},
```
